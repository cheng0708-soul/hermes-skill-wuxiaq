# 原版武侠Q传攻防公式破解记录

> 日期：2026-06-06

## 一、普攻伤害公式

```
最终伤害 ≈ atk² / (atk + def × C) + 附加伤害
```

其中 C 为版本修正系数，与关卡难度或角色等级相关。

特点：
- 非减法公式（不是 atk - def）
- 非纯百分比公式（不是 def/(def+C)）
- 是 **atk与def的比例关系**——atk越大，def的折算效果越被稀释
- 防御再高也不能完全免疫（但可以压到很低）

## 二、技能伤害公式

```
技能基础威力 = atk × 武功攻击加成 
             + mp × 武功内力加成 
             + spd × 武功身法加成

技能最终伤害 ≈ 基础威力 × (1 - 减伤率) - 化解值
```

### 关键：内力不参与百分比减伤

内力独立提供一个 **化解值**——直接减固定数值的伤害，不参与百分比计算。
高内力→高化解→极大抵消武功伤害。

## 三、二阶属性的影响

| 属性 | 效果 |
|------|------|
| 暴击 | 伤害 × 1.5 |
| 抗暴 | 降低被暴击概率 |
| 破招 > 格挡 | 防守方防御力大幅贬值（破甲效果） |
| 格挡 > 破招 | 格挡触发，最终伤害减半 |
