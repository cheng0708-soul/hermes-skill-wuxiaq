## 核心公式

```
实际触发率 = 面板施展几率 + 最终身法 / 1000 × 身法系数
```

- **面板施展几率**：武功自带的固定值（15-38%），原版故意写低
- **身法系数**：暂定1.0（可调0.6-1.0）
- 身法低的角色（段誉spd≈330）：加成+33%
- 身法极高的角色（东方spd≈705）：加成+70.5%

**例：令狐冲（60级，spd=294，独孤九剑面板22%）**
```
实际 = 22% + 294/1000 × 1.0 = 22% + 29.4% = 51.4%
```
→ 解释了"面板22%但实战体感高得多"
