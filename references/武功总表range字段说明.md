# 武功总表 range 字段说明

> 状态：定稿 ✓（2026-06-07 勘误后）
> 参考：原版玩家社区拆包 + 用户确认

---

## 一、字段格式

```
range = "<范围前缀><百分比系数>"
```

示例：
```
"贯穿63"  → range_type="贯穿", range_multiplier=63
"横扫18"  → range_type="横扫", range_multiplier=18
"单体102" → range_type="单体", range_multiplier=102
"全体15"  → range_type="全体", range_multiplier=15
"溅射22"  → range_type="溅射", range_multiplier=22
```

## 二、范围前缀 → AOE 模式

| 前缀 | 目标数 | 说明 |
|------|--------|------|
| 单体 | 1个 | 当前正对目标 |
| 贯穿 | 1列（≤3个） | 纵向穿透，如六脉神剑 |
| 横扫 | 1排（≤3个） | 横向攻击，如降龙十八掌 |
| 溅射 | 目标+周围（≤5个） | 中心目标+附近 |
| 全体 | 全部（≤6个） | 全员伤害 |

## 三、百分比系数 → 威力乘数

技能基础威力的计算公式：

```
技能基础 = (atk×aw + mp×mw + spd×sw) × (range_multiplier / 100)
```

其中 range_multiplier = range 字段的数字部分。

## 四、AOE 位置衰减

当技能命中多个目标时，按命中顺序引入衰减系数：

| 目标顺序 | 衰减系数 | 说明 |
|---------|---------|------|
| 第1个 | 100% | 全额伤害 |
| 第2个 | 80% | 衰减20% |
| 第3个 | 60% | 衰减40% |
| 第4个+ | 50%→递减 | 极少出现 |

每个目标**各自独立**用自己的 def 和 mp 计算减伤和化解。即：
```
for each target in hit list:
    decayed_base = skill_base × position_decay[i]
    damage = calc_damage(decayed_base, target.def)
    damage = resolve_dispel(damage, attacker.mp, target.mp)
```

## 五、实战对比（60级全输出）

| 技能 | 系数 | 单目标伤害 | 三目标总伤 |
|------|------|-----------|-----------|
| 段誉·六脉神剑(贯穿63%) | 63% | 231 | 495(231+163+101) |
| 萧峰·降龙十八掌(横扫18%) | 18% | 24 | 50(24+16+9) |
| 令狐冲·独孤九剑(单体88%) | 88% | 144 | 144(单体) |

## 六、常见误解

- ❌ **贯穿=破防** → 不是。贯穿是打一列，没有无视防御的机制
- ❌ **数字=破防值** → 不是。63是63%，不是无视63点防御
- ❌ **数字=威力指数，越大触发率越低** → 不是。range数字和触发率无关
- ✅ **正确**：数字是百分比系数，仅用来算技能基础威力
