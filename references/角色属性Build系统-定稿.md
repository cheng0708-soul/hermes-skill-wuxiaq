```
│                                                       │
│  ② 数值层（Balance Tuning）   ← 随时可以调             │
│     攻防公式 C=3、自由点收益、化解系数、暴击率           │
│     文档位置：§三 / §四                                │
│                                                       │
│  ③ 资源层（Resource Cycle）   ← 节奏控制               │
│     真气/冷却/触发率、大招消耗、战斗节奏                 │
│     文档位置：§五 / §十                                │
│                                                       │
│  ④ 策略层（Tactical Layer）   ← 玩家操作空间            │
│     阵容搭配、流派克制、站位策略、缘分体系               │
│     文档位置：§十二 → 【本节为定稿补入】                │
└─────────────────────────────────────────────────────┘
```

### 层间耦合说明（重要）

| 如果你改 | 会影响 | 需要同时检查 |
|---------|--------|------------|
| 前排挡刀规则（规则层） | 数值层的坦克生存期望 | 防御常数的平衡校验 |
| 防御常数C（数值层） | 资源层的战斗回合数 | 触发率节奏设计 |
| 自由点收益（数值层） | 策略层的流派克制关系 | 全防vs全血的平衡 |
| 真气获取速度（资源层） | 策略层的大招释放频率 | 阵容强度评估 |
| 触发率（资源层） | 策略层的核心角色定位 | 预期战斗回合数 |

**原则**：改一层的同时，至少检视相邻两层的影响，避免连锁失衡。

## 一、核心设计哲学

### 顶层战略：我们的核心竞争力是「阵容搭配」**

```
梦幻西游：12门派 × 固定属性 = 12种模板
  → 门派少 → 要靠堆系统制造多样性
  → 所以有：修炼/经脉/装备/宝石/宠物/坐骑/法宝...

武侠Q传：64角色 × 独特atk:mp:spd:def = 64种天生不同的角色
  → 人物多 → 排列组合本身就够玩了
  → 不需要照搬梦幻那一大堆系统
```

**关键决策原则：**
- 64角色的排列组合（6人队×9宫格×自由加点）才是核心护城河
- 加太多系统稀释"人物多"这个优势
- 只做最必要的系统：**武功修炼（P0）**，装备简化（P1），不做天赋树和宠物

### 三条落地原则

1. **数据库不动**：角色 JSON 的 base/growth 五属性（hp/atk/mp/spd/def）不改
2. **玩家有选择权**：每升1级给自由点，玩家决定角色成长方向
3. **三维定位鲜明**：同一个角色，不同加点 = 不同定位

### 角色定位三角

```
          武力(外功型)
         /        \
  速度快          输出高
   │               │
   │    ┌──────┐   │
   └────┤ 均衡 ├───┘
        └──────┘
   │               │
  防御高          内力厚
         \        /
          内力(内功型)
```

- **外功型**（西门吹雪、阿飞）：武力高、速度快 → 出手快、物理爆发
- **内功型**（段誉、无涯子）：内力高、内力辅助防御 → 站得住、内功爆发
- **均衡型**（萧峰、令狐冲）：武力内力均衡 → 灵活多变、适配多种武功

## 三、升级模型 【数值层】

### 3.1 每级收益

```
固定成长：base + growth × (level - 1)
自由加点：每升1级获得 2 自由点
```

### 3.2 自由点效果

| 投入项 | 每点收益 | 战略价值 |
|--------|---------|---------|
| 武力 | +3 atk | 物伤++ |
| 内力 | +3 mp | 内伤++，捎带硬一点 |
| 身法 | +2 spd | 抢先手 |
| 防御 | +5 def | 物减免 |
| 气血 | +10 hp | 血厚 |

### 3.3 数值平衡校验

以令狐冲 vs 萧峰（60级满级）为基准：

```
无自由点时：1.4击击杀          ← 原版平衡基准
全点武力：  1.2击击杀           ← 感觉得到变强
全点气血：  3.7击才被击杀        ← 血牛路线成立
全点防御：  1.9击才被击杀        ← 硬了一点
```

2点/级的推演：
- 即使极端全点武力，1.0击≈刚好不能秒人
- 正常玩家分散加点，比原版强10-20%
- 配合未来装备缘分→足够有养成感，但不崩

为什么选2点而不是1点：
- 1点/级 → 加点像"小点缀"，玩家感受不到存在
- 2点/级 → 加点值得认真思考，"每级一个决策"
- 等于原版自然成长的30%-60%，刚好是"有存在感的配角"

## 四、攻防公式 【数值层】（2026-06-06 基于原版拆包公式定稿）

### 4.1 普攻公式

```
普攻伤害 = atk² / (atk + def × C)

推导：普攻伤害 = atk × (1 - 减伤率)
     减伤率 = def×C / (def×C + atk)
```

| 参数 | 值 | 说明 |
|------|-----|------|
| C | 3 | 防御常数，可平衡微调 |

### 4.2 技能公式

```
技能基础威力 = atk × 武功攻击加成
             + mp × 武功内力加成
             + spd × 武功身法加成

技能减伤率 = def×C / (def×C + 技能基础威力)

技能最终伤害 ≈ 技能基础威力 × (1 - 技能减伤率) - 化解值
```

注：技能减伤率与普攻减伤率结构一致（`def×C/(def×C+攻击力)`），
区别在于普攻以 atk 为攻击力，技能以 技能基础威力 为攻击力。

### 4.3 化解机制（概率触发，仅对技能生效）

**化解触发概率：**
```
化解触发率 = 防守方mp / (防守方mp + 攻击方mp) × 基础概率
基础概率 = 0.8（默认，范围0.7-0.85）
```
- 普攻无法被化解
- 防守方mp远高于攻击方mp时：触发率接近80%（不是100%）
- 双方mp相等时：触发率 = 50% × 0.8 = 40%
- 防守方mp被压制时：触发率接近0
- 留15%-30%的漏招空间，让高威力武功仍有命中机会

**化解值（触发后生效）：**
```
化解值 = 面板固定化解（装备/阵法提供）
       + (防守方mp - 攻击方mp) × 内力化解系数
内力化解系数 = 0.2（默认，范围0.15-0.25）
```
- 内力被压制时（防守mp < 攻击mp），差值部分归零，只靠装备保底
- 示例：防守方mp比攻击方高500 → 化解值 +100

### 4.4 公式特点

**普攻 vs 技能的关系：**

| 角色类型 | 普攻强度 | 技能强度 | 原因 |
|---------|---------|---------|------|
| 外功型（西门） | 强 | 更强 | 技能基础 > atk |
| 均衡型（令狐冲） | 中等 | 略强 | 技能基础 ≈ atk |
| 内功型（段誉） | 极弱 | 爆炸 | 技能基础 >> atk |

**内力化解的价值：**
- 高内力角色面对技能时有天然减免（化解）
- 低内力角色（如西门mp=0）面对技能时几乎无化解
- 但外功角色的普攻不受化解影响 → 外功打高mp的对手时用普攻更划算

### 4.5 60级极限测试（C=3, def=+5, hp=+10）

| 场景 | 普攻 | 技能 | 结论 |
|------|------|------|------|
| 令狐冲(全武)→萧峰(全防) | 活4.3击 | 活3.8击 | 坦克守得住 ✅ |
| 令狐冲(全武)→萧峰(无点) | 活1.9击 | 活1.4击 | 无点合理 ✅ |
| 令狐冲(全武)→段誉(全内) | 活1.1击 | 活0.9击 | 脆皮该死 ✅ |
| 段誉(全内)→萧峰(全防) | 活17.6击 | 活3.2击 | 内功靠技能不靠普攻 ✅ |

**全防 vs 全血平衡（def=+5, hp=+10）：**
- 萧峰全防 4.3击 / 全血 4.4击 → 几乎持平 ✅
- 全防面对技能时有化解加持 → 综合更优 ✅

### 4.6 二阶属性修正

| 属性 | 效果 |
|------|------|
| 暴击 | 最终伤害 × 1.5 |
| 抗暴 | 降低被暴击概率 |
| 破招 > 格挡 | 防御方防御力大幅贬值（破甲） |
| 格挡 > 破招 | 格挡触发，伤害减半 |

## 五、武功数据结构 【资源层】

### 5.1 武功权重

每个武功包含以下字段：

```json
{
  "name": "独孤九剑",
  "type": "外功",
  "quality": "甲",
  "range": "单体31",
  "cost": 30,
  "target": "single",
  "atk_weight": 0.7,
  "mp_weight": 0.3,
  "spd_weight": 0.0,
  "trigger_rate": 55
}
```

| 新增字段 | 说明 |
|---------|------|
| `trigger_rate` | 技能主动触发概率（%），真气充足时每回合判定一次 |
| `type` | 外功/内功/轻功/心法——影响触发率基准值 |
| `range` | 原版威力数值+范围类型，如"单体31""贯穿63" |

### 5.2 触发率机制

> **来源**：原版武侠Q传武功配置表的"施展几率"字段。
> **状态**：定稿 ✓（2026-06-08 基于85人完整数据重写）
> **⚠️ 旧版(2026-06-07)的"外功55-65%/内功35-50%"为AI推断值，全部无效。**

**核心规则**：
- 每个武功有**面板施展几率**（15-38%），位于技能配置表
- 实际触发时叠加**身法加成**：`实际 = 面板 + (最终spd / 1000) × 身法系数`
- 身法系数暂定 **1.0**（可调参数）
- 真气达标才有资格触发，触发后消耗真气
- 心法类（13个被动）trigger_rate = 100%，不参与判定

**实际触发率公式**：
```
实际触发率 = 面板施展几率 + (角色最终spd / 1000) × 身法系数
```
例：令狐冲面板22%，60级spd=294，身法系数1.0
→ 实际 = 22% + 294/1000 = **51.4%**

**面板施展几率分级**（不是最终值）：

| 面板范围 | 代表角色 | 实战触发（60级） |
|---------|---------|----------------|
| 35-38%（面板已高） | 阿飞/李寻欢/虚竹/玄苦 | 55-93% |
| 28-32% | 周伯通/小龙女/苗人凤 | 45-70% |
| 22-25% | 令狐冲/段誉/叶孤城/萧峰 | 45-55% |
| 15-20% | 东方不败/洪七公/谢逊 | 30-88%（spd决定） |

**核心角色一览**：

| 角色 | 天赋武功 | 面板 | 范围 | 60级spd | 实战触发 | 期望节奏 |
|------|---------|------|------|---------|---------|---------|

**完整数据详见**：`references/天赋武功总表.json`（85人完整JSON）和 `references/天赋武功总表.md`（Markdown版）

**面板 vs 实战的关键**：
- 身法越高，面板触发率的"水分"越大
- 东方不败面板仅18%，但705身法补正到88.5%
- 萧峰面板18%，275身法仅补到45.5%
- **评估角色强弱必须看实战触发率，不是面板值**

**身法系 vs 内功系对比**：
```
身法流（东方/楚留香/李寻欢）：面板低（18-35%）+ 高身法 → 实战极高（88-96%）
内功流（段誉/虚竹/一灯）：      面板中（22-35%）+ 中等身法 → 实战适中（50-70%）
重装流（萧峰/郭靖/洪七公）：    面板低（15-18%）+ 低身法 → 实战偏低（35-50%）
---------------------------------------
结论：身法系才是真正的高频触发者，不是外功系
```

**心法处理**：
13个被动心法 trigger_rate = 100%（不参与触发判定），常驻生效。
详见 `references/天赋武功总表.md → 第四章 被动心法总表`

### 5.3 完整伤害链

```
每个回合开始时：

① 真气积累
   真气 += 10(本回合普攻) + 5(本回合受击)

② 触发判定（决定本回合放技能还是普攻）
   if 真气 ≥ 技能消耗 且  rand < trigger_rate:
       → 触发！释放技能（走③），真气 -= 技能消耗
   else:
       → 未触发，本回合改为普攻（走②普攻分支）

[技能分支]
③ 最终属性 = base + growth×(Lv-1) + 自由点加成
   最终atk = base_atk + growth_atk×(Lv-1) + 自由武力点×3
   最终mp  = base_mp  + growth_mp ×(Lv-1) + 自由内力点×3
   最终def = base_def + growth_def×(Lv-1) + 自由防御点×5
   最终spd = base_spd + growth_spd×(Lv-1) + 自由身法点×2
   最终hp  = base_hp  + growth_hp ×(Lv-1) + 自由气血点×10

④ 技能基础 = (最终atk × atk_weight + 最终mp × mp_weight + 最终spd × spd_weight)
          × (range_multiplier / 100)

    AOE类型：单体打1人、贯穿打一列、横扫打一排、溅射打周围、全体打全部

    真气消耗：每个技能配置表中的固定整型值（如技能表的 MpCost/SpCost 字段）。
    由策划手动填写，不按 range_multiplier 比例自动计算。
    群攻技能（横扫/贯穿/全体）的固定消耗通常比单体高，以达到平衡。

   技能原伤 = 技能基础² / (技能基础 + 最终def × 3)

⑤ AOE 位置衰减（作用于技能原伤，逐个目标独立计算）：

   当技能命中的目标数 ≥ 2 时（贯穿/横扫/溅射/全体）：
     第1个目标不受衰减：技能原伤 × 100%
     第2个目标衰减20%：技能原伤 × 80%
     第3个目标衰减40%：技能原伤 × 60%
     每个目标分别用自己的 def 和 mp 独立计算减伤和化解
     （即 3 次独立的"攻击 vs 防御/化解"公式计算）

⑥ 化解判定（仅技能生效，普攻不化解）：
   触发率 = 防守方mp / (防守方mp + 攻击方mp) × 0.8
   化解值 = 面板化解 + max(防守方mp - 攻击方mp, 0) × 0.2
   if rand < 触发率: 技能终伤 = 技能原伤(衰减后) - 化解值
   else: 技能终伤 = 技能原伤(衰减后)

[普攻分支]
②' 普攻伤害 = 最终atk² / (最终atk + 最终def × 3)

[统一收尾]
⑥ 暴击判定（技能和普攻均适用）：
   if rand < 0.15: 伤害 ×= 1.5

⑦ 缘分加成：伤害 × (1 + 缘分百分比之和)
⑧ 阵位加成：伤害 × (1 + 格子属性加成%)
⑨ 随机波动：伤害 × rand(0.95, 1.05)
⑩ 最终伤害 = max(1, round(伤害))
```

### 5.4 V1→V2 迁移

- V1的 `powerStat: "atk"` → V2的 `atk_weight:1.0, mp_weight:0`
- V1的 `powerStat: "mp"` → V2的 `atk_weight:0, mp_weight:1.0`
- V2新增双权重（混合型武功）——不是破坏性变更

## 六、9宫格与Build联动 【规则层】

### 6.1 格子加成

| 阵位 | 加成 | 适合哪种加点 |

## 十、9宫格战斗规则 【规则层 / 资源层混合】

### 10.1 普攻规则（2026-06-06定）

```
优先打同列最前排 → 前排不倒打不到后排
前排角色在保护后排的同时自己也承担了伤害
强力角色放前排=浪费输出潜力（取舍）
```

### 10.2 技能目标规则

```
单体技能：可打任意目标（包括后排）
贯穿技能：打一整列（前后一起打）—— 勘误：原版「贯穿」=破防，非穿排
横扫技能：打一整行
溅射技能：打目标及其周围
全体技能：打所有目标
```

### 10.3 真气/能量系统

```
开局真气：30
每回合普攻：+10
每回合被击：+5
技能消耗：小技能20-30 / 主技能40-60 / 大招70-100
```

### 10.4 战斗节奏

```
第1回合：普攻对轰前排 + DPS小技能切后排（有开局真气）

第3-5回合：对轰期 — 哪边后排先倒哪边崩盘
```

## 十二、策略层（Tactical Layer）——阵容与流派克制 【本轮定稿补入】

> **状态**：定稿 ✓（2026-06-07）
> **核心观点**：64角色的排列组合（6人队×9宫格×自由加点×流派克制）才是真正的核心竞争力。策略层是玩家"操作空间"的体现——阵容对了，低战力也能打赢。

### 12.1 四大流派的克制关系

```
刚猛外功（金/橙）─── 克 ───→ 暗器毒杀（绿/紫）
       │                            │
       │ 被克                        │ 克
       ↓                            ↓
内功奇门（蓝/紫）─── 克 ───→ 回春辅助（翠绿）
```

**克制效果**：
- 攻击克制方时，**最终伤害 × 1.2**
- 攻击被克方时，**最终伤害 × 0.85**
- 同流派对战时，**无加成**

**流派克制设计意图**：
| 克制关系 | 设计理由 |
|---------|---------|
| 刚猛→暗器 | 暗器速度快但脆，重装硬打就是克星 |
| 暗器→回春 | 暗器切后排，治疗角色站不住 |
| 回春→内功 | 续航拖死爆发流，内功打不穿治疗链 |
| 内功→刚猛 | 内功无视化解（高mp），刚猛防御不全是物理 |
| 刚猛→内功（被克） | 刚猛mp低→被内功化解减伤，打不出伤害 |
| 暗器→刚猛（被克） | 暗器被刚猛硬突到脸上就废 |

### 12.2 阵容站位策略（给玩家的建议）

**前排（承受位）**：
```
推荐：刚猛外功型 / 高防均衡型
加点方向：防御+气血
如：萧峰（全防）、郭靖（全血）、谢逊（全防）
目的：扛住第一波伤害，保护后排输出
```

**中排（核心位）**：
```
推荐：内功奇门型 / 均衡型
加点方向：自由分配（看队伍缺哪边）
如：段誉（全内）、杨过（均衡）、令狐冲（全武）
目的：承担主要输出
```

**后排（脆弱位）**：
```
推荐：暗器毒杀型 / 回春辅助型 / 高攻脆皮
加点方向：身法+武力或身法+内力
如：东方不败（全速）、小龙女（均衡）、楚留香（全速）
目的：优先出手 / 治疗 / 补刀
```

### 12.5 策略层的设计原则

1. **不要做死套路**：玩家应该有自由空间，克制关系给 1.2x/0.85x 就行，不用搞成"暗器打不过刚猛"的死循环
2. **缘分为辅不为主**：缘分加成不超过 30-50%，不让"凑缘分"代替"建阵容"
3. **站位要有取舍**：强力角色放前排浪费输出潜力但保护了后排→每局都要权衡
4. **自由点加深差异**：同一个人物全武 vs 全防是两个完全不同的定位→可重复培养

### 12.6 不做清单（策略层限定）

```
❌ 布阵加水/火/风地形加成（太复杂，不适合本游戏节奏）
❌ 羁绊树强制绑定（玩家想自由配队，不要搞"不上XX就不能玩"）
❌ 对战预判/读心机制（回合制卡牌，不需要搞心理博弈）
✅ 做：简单的流派克制 + 缘分激励 + 站位取舍
```
