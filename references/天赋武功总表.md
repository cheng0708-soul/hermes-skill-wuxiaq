## 核心机制

### 技能威力公式

```
武功附加威力 = 最终atk × atk_ratio + 最终mp × mp_ratio + 最终spd × spd_ratio
全额理论伤害 = 附加威力 × (range_multiplier / 100)
最终伤害 ≈ 全额理论伤害² / (全额理论伤害 + def × 3) - 化解值
```

`attr = [atk_ratio, mp_ratio, spd_ratio]` 对应属性加成比例（武力:内力:身法）

### 实际触发率公式

```
实际触发率 = 面板施展几率 + (最终spd / 1000) × 身法系数
身法系数 = 1.0（可调参数，待定稿确认）
```

## 六、实战触发率补正表

以下是核心角色的**面板施展→实战最终触发率**（系数=1.0）：

| 角色 | 面板 | 60级spd | 身法补正 | 实战触发 | 期望回合 |
|------|------|---------|---------|---------|---------|
| **阿飞** | 38% | ~180 | +18% | **56%** | 1.8 |
| **东方不败** | 18% | **705** | +70.5% | **88.5%** | 1.1 |
| **楚留香** | 30% | ~660 | +66% | **96%** | 1.0 |←几乎必放 |
| **李寻欢** | 35% | ~580 | +58% | **93%** | 1.1 |
| **西门吹雪** | — | **561**(被动爆炸) | 被动 | 被动 | 被动 |
| **令狐冲** | 22% | 294 | +29.4% | **51.4%** | 1.9 |
| **叶孤城** | 25% | ~450 | +45% | **70%** | 1.4 |
| **萧峰** | 18% | 275 | +27.5% | **45.5%** | 2.2 |
| **段誉** | 22% | 330 | +33% | **55%** | 1.8 |
| **虚竹** | 35% | ~330 | +33% | **68%** | 1.5 |

**关键发现**：身法系角色（东方、楚留香、李寻欢、叶孤城）面板虽低（18-35%），但超高spd补正后实战触发率远超面板——这解释了原版"面板22%但体感高很多"的现象。内功角色（段誉、虚竹）spd中等，补正幅度10-30%。

## 七、与前版文档的差异说明

| 项 | 旧值 | 新值 | 原因 |
|----|------|------|------|
| 独孤九剑触发率 | 55% | **22%** | 旧版误用推导公式，这是面板值 |
| 触发率公式 | 固定值 | **面板+spd/1000×系数** | 身法影响实战触发 |
| 外功触发率范围 | 55-65% | **15-38%**（面板） | 原版实际数据 |
| 降龙十八掌触发率 | 55% | **18%**（面板） | 原版实际数据 |
| 段誉六脉触发率 | 42% | **22%**（面板） | 原版实际数据，但实战约55% |
| spd加权公式 | 无 | **atk:mp:spd三权重** | 原版 attr_add_rate 机制 |

> **⚠️ 旧版文档中所有 trigger_rate 值（55-65%等）均无效，请以本表为准。**
