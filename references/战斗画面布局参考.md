# 武侠Q传 战斗画面布局参考

> 来源：成哥提供的原版《武侠Q传》战斗截图描述
> 风格：经典 Q 版卡牌回合制武侠手游

---

## 整体氛围

国风比武场擂台，3D Q版二头身角色，左右对立式多角色卡牌回合制布局。

---

## 1. 顶部信息栏（对决全局状态）

| 元素 | 位置 | 描述 |
|------|------|------|
| **战斗回合标题** | 正中央上方 | 书法艺术字"战斗回合"，水墨或金属质感 |
| **己方信息** | 左上角 | 头像 + 昵称（如"傲.清扬"），头像框紫色高亮 |
| **敌方信息** | 右上角 | 头像 + 昵称（如"傲.剑魔"） |
| **出战人数** | 昵称内侧 | "出战 7/7" 数字条，实时显示场上存活角色数 |
| **替补人数** | 出战人数下方 | "替补 2/2" 字样，提示后援人数 |

---

## 2. 中部核心战场（经典卡牌对阵布局）

### 擂台
- 画面中心：巨大圆形红色地毯比武台
- 所有角色在台上进行回合制乱斗

### 阵营站位

```
┌─────────────────────────────────────────┐
│  [顶部信息栏]                              │
│                                           │
│   己方阵型          敌方阵型               │
│   ○ ○ ○            ○ ○ ○                 │
│    ○ ○              ○ ○                   │
│     ○                ○                     │
│    (6-7人)          (6-7人)               │
│   前排/中排/后排    前排/中排/后排          │
│                                           │
│  ┌───── 圆形红色地毯比武台 ─────┐          │
└─────────────────────────────────────────┘
```

- **左侧阵营（己方/主视角）**：6-7人，错落站位，分前排/中排/后排传统卡牌矩阵
- **右侧阵营（敌方）**：镜像对称，面对面
- **每个角色头顶**：红色血条（HP Bar），直观显示生存状态
- **角色造型**：大头小身 3D Q版二头身，古风武侠服饰（长袍/盔甲），手持长剑/大刀等

---

## 3. 数值飘字与视觉特效（动态反馈）

| 元素 | 描述 |
|------|------|
| **绿色加血飘字** | 如 `+33920`，绿色 + 黑边勾勒，巨大数字，出现在受治疗角色身上 |
| **脚底光环** | 不同颜色半透明光环环绕脚底，表示技能释放或被动buff触发 |
| - 青绿色光环 | 左下角角色 |
| - 粉紫色 + 蓝色流光 | 右下角角色 |

### 背景
- 古代武林擂台背景
- 两排锋利尖枪兵器架
- 两侧各一面金色战鼓（龙纹/祥云图案）
- 两侧挂大红灯笼
- 氛围：宗门大比/武林决战

---

## 4. 总结

这是一个极其标准的**左右对立式、多角色卡牌回合制画面布局**。通过：
- **大字飘数** → 实时传达伤害/治疗结果
- **血条缩减** → 直观展示生存状态
- **脚底特效** → 区分属性状态/被动buff

向玩家传递每一步回合交锋的战果。
