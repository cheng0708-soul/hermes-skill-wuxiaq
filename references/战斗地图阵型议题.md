## 核心议题

6人队战斗，用 **9宫格（3×3）放6人** 好，还是 **2排×3列固定6坑** 好？

---

## 两种方案对比

| 维度 | 2排×3列（6坑满） | 9宫格（3×3放6人） |
|------|------------------|------------------|
| 布阵排列数 | 720种（6!） | 60,480种（C(9,6) × 6!） |
| 策略深度 | ★★ 只有谁站哪 | ★★★★ 空哪格+谁站哪 |
| 新手理解 | ★★★★ 一目了然 | ★★★ 格子概念稍多 |
| 文化包装 | 一般 | 强（八卦阵/锋矢阵/七星阵） |
| 克制系统 | 难做 | 天然（阵型相克） |
| UI复杂度 | 低 | 中（需格子拖拽+状态显示） |
| 空位战术 | ❌ 无 | ✅ 故意空格诱敌/陷阱 |

## 9宫格推荐方案

### 格子布局

```
    左侧（己方）        右侧（敌方）
    ┌──┬──┬──┐      ┌──┬──┬──┐
前   │  │  │  │      │  │  │  │
    ├──┼──┼──┤      ├──┼──┼──┤
中   │  │  │  │  VS  │  │  │  │
    ├──┼──┼──┤      ├──┼──┼──┤
后   │  │  │  │      │  │  │  │
    └──┴──┴──┘      └──┴──┴──┘
     左 中 右         左 中 右
```

### 格子属性加成方案（待定）

| 格子 | 位置 | 建议加成 |
|------|------|---------|
| 前1/前3 | 前排边路 | +10% 闪避/格挡 |
| 前2 | 前排中路 | +15% 减伤 |
| 中1/中3 | 中排边路 | +8% 暴击率 |
| 中2 | 中排中路 | +10% 技能伤害 |
| 后1/后3 | 后排边路 | +8% 暴击伤害 |
| 后2 | 后排中路 | +12% 治疗效果 |

### 空位机制

空位策略是9宫格的核心亮点：

- **故意空前排**：让敌方刺客切进来，但我方后排安排了反杀型
- **故意空后排**：保护后排不被打，但被切后排技能打到时无人替死
- **布局成特定阵型**：触发阵型buff

### 阵型名称 & 克制（画龙点睛的包装）

| 阵型 | 布法 | 克制 | 被克 |
|------|------|------|------|
| **锋矢阵** | 前排3人满，后排只放中路 | 雁行阵 | 八卦阵 |
| **雁行阵** | 斜线布阵 | 太极阵 | 锋矢阵 |
| **八卦阵** | 均匀分布（每行2人） | 锋矢阵 | 雁行阵 |
| **七星阵** | 仿北斗7星排列 | 单体阵容 | 群伤阵容 |
| **一字长蛇** | 全部站中间列 | 面伤阵容 | 列贯穿阵容 |
