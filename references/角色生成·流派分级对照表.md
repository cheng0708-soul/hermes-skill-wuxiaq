## 六、飞弹形状——角色映射矩阵

角色生成器的关键产出：**决定每个角色用哪种飞弹形状**

| 飞弹形状 | 对应 Sprite 需求 | 适用角色 |
|---------|----------------|---------|
| **掌印**（盾形/圆形） | 需画 3 帧金色透明掌印 | 萧峰、虚竹、张无忌、铁中棠、萧远山、玄慈、谢逊（音波波） |
| **月牙剑气**（弯月形） | 需画 3 帧白色/灰色月牙 | 西门吹雪、令狐冲、步惊云、苗人凤、杨过、叶开 |
| **太极球**（旋转圆形） | 需画 3 帧蓝/紫色旋转光球 | 段誉（偏蓝）、无崖子（偏紫）、李秋水（偏白） |
| **指芒**（细长光束） | 需画 2 帧金色/蓝色细线 | 一灯大师、段正淳、段延庆、黄眉僧、慕容博、鸠摩智 |
| **飞针**（极细短镖） | 需画 2 帧黄/白色细针 | 东方不败、练霓裳、上官小仙（飞刀） |
| **刀风**（弧形宽波） | 需画 3 帧白/金色宽弧 | 胡一刀、胡斐、血刀门主、金轮法王 |
| **音波**（扩散波纹） | 需画 2 帧圆形波 | 谢逊、黄药师、任盈盈 |

**统计**：7 种飞弹 × 每种 2-3 帧 = **约 18 帧像素 Sprite** 即可覆盖全部 64 角色。

## 九、给角色生成器 AI 的操作流程

```
① 查本表 → 确定角色流派、武器类型、飞弹形状
② 04_技能图标 → 选对应的图标文件夹（Assassin/Mage/Barbarian/Archer）
③ 05_装备与道具图标 → 选装备图标（刀剑/药水/书籍/铠甲）
④ 填 JSON：class + weapon_type + projectile_shape
⑤ 生成 spum_parts（部件号+色码）
⑥ 主 AI 会通过特效系统自动匹配颜色和粒子
```

**一句话**：做角色时只要告诉我 `class`、`weapon_type`、`projectile_shape` 三个字段，特效全自动。
