# 人物升级与战斗系统设计

> 参考：梦幻西游回合制 + 9宫格阵型
> 状态：待定稿 / P0

---

## 一、核心理念

**梦幻西游式自由加点 + 武侠Q传人物特色 + 9宫格阵型策略**

```
梦幻西游：每级5自由点 + 5属性 → 衍生所有战斗数值
    ↓
武侠Q传：每个英雄有天生偏向（外功/内功/均衡/敏捷/防御）
    ↓
9宫格：不同格子加成不同属性倾向
```

## 二、五大基础属性

### 2.1 属性定义

| 属性 | 代号 | 对应武侠概念 | 影响 |
|------|------|-------------|------|
| **体质** | CON | 气血/根骨 | 气血上限 + 少量防御 |
| **内力** | INT | 丹田/真气 | 真气上限 + 技能伤害(内) + 内功防御 |
| **力道** | STR | 臂力/劲力 | 物理伤害 + 少量速度 |
| **根骨** | END | 筋骨/体魄 | 物理防御 + 少量气血 |
| **身法** | AGI | 轻功/步法 | 速度 + 闪避 + 暴击 |

### 2.2 为什么用这五属性（而不是梦幻的体魔力耐敏）

梦幻西游的「魔力」在武侠世界观中不成立（武侠不靠"魔力"），替换为 **内力**：
- 梦幻的魔力 = MP + 法伤 + 法防 → 武侠的内力 = 真气 + 内功伤害 + 内功防御
- 梦幻的力量 → 武侠的力道（更古风）
- 梦幻的耐力 → 武侠的根骨
- 梦幻的敏捷 → 武侠的身法
- 体质保留

这样既能套用梦幻的加点机制，又**不违和中国武侠味**。

## 三、衍生属性计算公式

### 3.1 战斗属性

```
气血 = CON × 15 + 等级 × 10
真气 = INT × 10 + 等级 × 5
物理攻击 = STR × 3 + 等级 × 2
物理防御 = END × 2.5 + 等级 × 1
内功攻击 = INT × 2.5 + 等级 × 1.5
内功防御 = INT × 1.5 + END × 1 + 等级 × 1
速度 = AGI × 2 + 等级 × 0.5
闪避 = AGI × 0.3 + 基础闪避
暴击 = STR × 0.2 + AGI × 0.1 + 基础暴击
会心 = STR × 0.15 + INT × 0.15  (暴击后的额外伤害系数)
命中 = 基础命中 + 等级 × 0.5
抗暴 = END × 0.2 + 基础抗暴
```

### 3.2 为什么用乘法系数（而不是加法）

系数可以 **区分人物定位**：
- 令狐冲的STR系数 = 3.5 → 每点力道多给0.5物攻 → 加点收益更高
- 段誉的INT系数 = 3.5 → 每点内力多给1.0内攻 → 加点收益更高
- 这样同一套公式，**不同人加点策略完全不同**

## 四、人物天生成长（每级固定 + 自由点）

### 4.1 每级增长

每个角色升级获得：
- **固定成长**：基础五属性自动涨（受人物品质和类型影响）
- **自由点**：5点可供玩家分配

### 4.2 人物类型 vs 成长系数

| 类型 | 代表人物 | CON成长 | INT成长 | STR成长 | END成长 | AGI成长 | 特色 |
|------|---------|---------|---------|---------|---------|---------|------|
| **外功型** | 萧峰、令狐冲 | 中 | 低 | **高** | 中 | 中 | STR成长最高，适合力道流派 |
| **内功型** | 段誉、一灯 | 中 | **高** | 低 | 中 | 中 | INT成长最高，适合内力流派 |
| **均衡型** | 楚留香 | 中 | 中 | 中 | 中 | 中 | 无明显短板 |
| **敏捷型** | 东方不败、田伯光 | 低 | 低 | 中 | 低 | **高** | AGI成长高，抢先手 |
| **防御型** | 扫地僧、张无忌 | **高** | 中 | 中 | **高** | 低 | CON+END成长高，铁板 |

### 4.3 品质影响基础值

```
甲级：初始五属性总和 ≈ 120-140
乙级：初始五属性总和 ≈ 90-110
丙级：初始五属性总和 ≈ 60-80
丁级：初始五属性总和 ≈ 30-50
```

## 五、加点策略（玩家的核心乐趣）

### 5.1 经典加点流派（参考梦幻）

以外功型令狐冲为例，玩家可以有多种加点方案：

| 流派 | 加点 | 效果 | 适合阵位 |
|------|------|------|---------|
| **全力道** | 5STR | 极致物理输出，脆皮 | 后1/后3（边路后排） |
| **力道+身法** | 3STR+2AGI | 高伤害+抢先手 | 中2（中排中路） |
| **力道+根骨** | 3STR+2END | 能打能扛 | 前2（前排中路） |
| **均衡** | 2STR+1END+1CON+1AGI | 万金油 | 前1/前3（前排边路） |
| **暴力流** | 4STR+1AGI | 赌暴击 | 后2（后排中路） |

### 5.2 为什么要点流派

同一角色 → 不同加点 → **不同定位 → 不同阵位选择**

玩家可以：
- 抽到一个令狐冲，练成全力道型 → 放后排当主C
- 抽到第二个令狐冲，练成力根型 → 放前排当副C
- 甚至全重新加点，改变定位

## 六、9宫格与属性的联动

### 6.1 格子加成

回到之前的设计，每个格子加微量属性百分比：

```
前排（挨打位）
  前1/前3: +5% 根骨（防御）
  前2:     +8% 体质（血量）

中排（过渡位）
  中1/中3: +5% 力道（物攻）
  中2:     +5% 内力（法攻）

后排（输出位）
  后1/后3: +5% 身法（速度+暴击）
  后2:     +5% 力道 或 +5% 内力（自选）
```

### 6.2 加点+阵位的联动策略

```
全力道令狐冲 → 放后1（+5%身法）→ 暴击流
力根令狐冲   → 放前2（+8%体质）→ 半肉
全内力段誉   → 放中2（+5%内力）→ 极致内伤
体根扫地僧   → 放前2（+8%体质）→ 铁壁
```

## 七、回合制战斗流程（梦幻西游式）

### 7.1 速度决定行动顺序

```
每回合开始：
  1. 所有单位按速度排序
  2. 从高到低依次行动
  3. 每人每回合行动1次（普攻或放技能）
```

### 7.2 普攻规则

```
普攻目标选择：
  1. 优先攻击同一列的最前排敌人
  2. 如果同列前排已空 → 打同列后排
  3. 如果同列全空 → 攻击相邻列
```

### 7.3 技能规则

```
技能类型：
  - single: 单体攻击（可指定目标）
  - row:    打整行（前排3人/中排3人/后排3人）
  - col:    打整列（同一列前后排）
  - all:    打全体
  - heal:   治疗（单体/全体）
  - buff:   增益（+攻/+防/+速）
  - debuff: 减益（降攻/减速/眩晕）
```

### 7.4 真气/怒气系统

```
普攻获得：+10真气
受击获得：+5真气
技能消耗：30-80真气（越强力消耗越多）
开局：0真气
```

## 十、总结：玩家的升级变强路径

```
① 抽到角色（甲级最好）
    ↓
② 升级（每级+5自由点，按流派加点）
    ↓
③ 配装备（武器/防具/饰品/秘籍）
    ↓
④ 凑缘分（特定队友+装备）
    ↓
⑤ 选阵位（9宫格 × 加点流派匹配）
    ↓
⑥ 上阵战斗
```

**每一条路径都是玩家可以主动选择和优化的 → 长期养成感**

## 十一、与当前系统的衔接

**当前 JSON 方案（stats/growth）需要改造为：**

```json
{
  "name": "令狐冲",
  "quality": "甲",
  "type": "外功型",
  "base": {
    "con": 18, "int": 12, "str": 25, "end": 15, "agi": 20
  },
  "growth": {
    "con": 1.2, "int": 0.8, "str": 2.0, "end": 1.0, "agi": 1.5
  },
  "skills": [
    { "name": "独孤九剑", "type": "single", "cost": 30, "power": 1.8, "scaling": "str" }
  ],
  "bonds": [
    { "name": "笑傲江湖", "members": ["任盈盈", "任我行"], "effect": { "str_pct": 0.15 } }
  ]
}
```

- `base` = 1级时的五属性
- `growth` = 每级固定增长（不消耗自由点）
- 自由点由玩家在游戏内分配（存到存档数据）
