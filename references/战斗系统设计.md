# 战斗系统设计

> **当前版本** = 可玩 MVP；本文记录已拍板的升级方向（2026-06-06定稿）。

---

## 1. 当前 MVP

| 特性 | 实现 |
|---|---|
| 战斗模式 | 自动回合制，按SPD排序轮流出手 |
| 队伍规模 | 玩家最多3人 vs 敌方N人 |
| 目标选择 | 全程AI自动（随机存活目标） |
| 玩家交互 | 仅：怒气满时点击按钮手动释放大招 |
| 阵型 | 单行横排，间距1.35f |
| VFX | 飘字+Log，无粒子特效 |
| 技能类型 | 普攻/大招，TargetMode=Single/All |
| 伤害计算 | powerStat区分物攻(ATK)/法攻(MP) |

---

## 2. 升级目标（已拍板）

### 2.1 六人阵型
- 双方各最多6人上场
- 阵型暂定单行延伸，保持1.35f间距
- 改GameSession.PlayerTeam扩容至6人

### 2.2 手动选目标
| 角色 | 方式 |
|---|---|
| 我方单位 | 玩家可手动指定，否则AI自动 |
| 敌方单位 | 始终AI自动 |

- 行动前可点击敌方预选目标（高亮框）
- 目标死后降级到AI自动

### 2.3 VFX策略
- 当前：继续飘字+Log占位
- 后期推荐：外包VFX。每个技能一个Prefab，名称`VFX_<skillId>`，放`Resources/VFX/`
